Q: Suppose A∈Mn and σ1(A)≤1, that is, A is a contraction. If H(A)≡12(A+A∗)=I, show that A=I.

A: First recall the lemma: A is normal iff A∗=AV for some unitary matrix V.

⟸: Since

[A∗=AV]⇔[A=V∗A∗]⇔[VA=A∗],

we see that A and V commutes. Hence

A∗A=(AV)A=A(VA)=AA∗.

⟹: If A is normal, then we have the decomposition A=U∗ΛU for unitary matrix U and diagonal matrix

Λ=diag(λ1,…,λn)=diag(|λ1|,…|λn|)⋅diag(eiθ1,…,eiθn)=:R⋅Θ.

Hence A=U∗RΘU, and

A∗=U∗Θ∗RU=U∗RΘUU∗(Θ∗)2U=A(U∗Θ∗Θ∗U).

Therefore, V=U∗Θ∗Θ∗U.

When A+A∗=2I, we have that

I=12R(Θ∗+Θ)=R⋅Re(Θ).

Since ∥Re(Θ)∥≤1, we learn that both R and Θ are {±1}n-valued. Now it is clear that RΘ=I.
