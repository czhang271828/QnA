Q: 如果多项式 f(x) 和 g(x) 的根都是实数: r1≤r2≤⋯≤rn 和 s1≤s2≤⋯≤sn−1, 满足 r1≤s1≤r2≤s2⋯≤sn−1≤rn, 则称 f 和 g 的根交错. 证明, 多项式 f, g 的根交错, 当且仅当对所有 α∈R, 线性组合 f+αg 的根都是实数.

A: 这是很显然的, 甚至可以按照下面的证明思路推广一下(例如有一处不交错等等). 以下证明是把简单的画图题用严谨的分析语言写出来, 从而字略多.

约定记号 C―:=C∪{∞}, R― 为 C― 的赤道面 R∪{∞}. 先做以下调整.

  • 不妨设恒有 ri≠sj, 允许 ri=∞. 例如 x 作为二次多项式, 其根为 {0,∞}.

  • 定义 < 为圆 R― 上的逆时针(C― 上俯视)相邻. 例如对集合 {1,4,−2,∞}, 有 1<4<∞<−2<1.

  • 令 g(x) 是 n 次全实根多项式, 满足 r1<s1<⋯<sn−1<rn. 将最后一句 "f+αg 的根都为实数"调整作"tf+(1−t)g 的根都在 R― 内, 其中 t∈[0,1]".

在 C― 上标注 g 的所有根 {si}1≤i≤n. 那么随 t 在 [0,1] 的变动, tf+(1−t)g 的 n 个的根在 C― 中表现为连续曲线 {γi(t)}1≤i≤n. 这些曲线满足以下几条性质.

  • 对 1≤i 考虑球面 C― 上通常意义的度量, 对任意 j 与 0≤t≤1, |γj′(t)| 被某一仅与 f 与 g 系数相关的常数固定(证明略去).

  • 由于 tf+(1−t)g 是实系数多项式, 故对任意 0≤t≤1, ⋃1≤i≤n{γi(t)} (n-点集合)关于赤道 R― 对称.

  • 对任意 i 与任意 t∈(0,1), γi(t) 不等于任何 g 与 f 的根; 反之可推得 f 与 g 有重根.

我们断言对任意 i≠j, 曲线 γi(t) 与 γj(t) 没有交点. 首先对于足够小的 ε>0, {γi(t)∣0≤t≤ε}1≤i≤n 扫过的经度两两不交, 根据 R―-对称性可知每一 {γi(t)∣0≤t≤ε} 均在 R― 上. 从而存在最大的 t0∈(0,1] 使得 {γi(t)∣0≤t≤t0} 均在 R― 上.

我们断言 t0=1. 若 t0<1, 则存在曲线 γi(t), 其轨道在 t>t0 时偏离 R―. 由于 ⋃1≤i≤n{γi(t0)} 关于 R―-对称, 故存在另一在 γi(t0) 处偏离 R― 的曲线 γj(t) . 注意到 t≤t0 时, γi(t) 均在 R― 上, 由交错性可知 γi 或 γj 在 0≤t≤t0 时的轨迹经过 f 的某一根, 矛盾!

以上证明了 t0=1, 亦即 {γi(t)∣0≤t≤1} 在 R― 上. 显然 γi 两两不交, 且起点相异以及终点相异, 因此对任意 1≤i≤n, γi 只能是 si 到 rj 的 R―-弧线, 或 si 到 rj−1 的 R―-弧线. 因此对任意 α∈R, f+αg 在 R― 上有 n 个相异的根.

原题中 sn=∞, f+αg 是 n 次多项式, 故在 R 上有 n 个两两不同的根.
