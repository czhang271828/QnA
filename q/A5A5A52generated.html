Q: A5×A5×A5 是 2-生成的吗?

A: A5 中的元素无非这几类: e, (ij)(kl), (ijk), (ijklm), 阶数分别是 1,2,3,5, 数量分别为 1,15,20,24. 那么判断

A5×A5×A5

2-生成与否就是个简单数论题目了. 例如取

a=(3 阶元)(5 阶元)(2 阶元),b=(2 阶元)(3 阶元)(5 阶元).

那么 a10=(3 阶元)ee, b15=(2 阶元)ee, 那么 a10 和 b15 生成 A5×{e}×{e}. 下略.

还要说明下

⟨(123),(12345)⟩=A5.

这里发现

(1,2,3,4,5)→(3,4,5,1,2)→(4,5,3,1,2)

的复合是 (14)(25), 根据主楼链接得

⟨(123),(12345)⟩=⟨(14)(25),(123),(12345)⟩=A5.

将上述过程反一下, 可以发现存在生成 5-阶元的 2-阶元与 3-阶元. 不妨设 a 的位置 1 与 b 的位置 1 取的就是这两个元.

Q: 是不是因为 2, 3, 5 只能组成 3 个互质的配对 (2,3), (2,5), (3,5)

a=(3 阶元)(5 阶元)(2 阶元),b=(2 阶元)(3 阶元)(5 阶元).

从这个例子看出: 每列, 每行元素需要两两互质.

A: 这个题目没这么简单. 我稍后做个具体的分析.

此后统一用 [k] 表示 k 阶元.

我们先尝试说明以下 ∏1≤k≤4A5 是 2-生成的. 例如取

1234生成元[2][2][3][5]⌊[3][5][2][2]⟹[1][1][1][5]⌊[1][1][2][2]⟹[1][1][1][5]⌊[1][1][2][3]⟹[1][1][1][5]⌊[1][1][1][3]

可知子群 {e}×{e}×{e}×A5 是 2-生成的. 根据对称性, ∏1≤k≤4A5 是 2-生成的. 我们断言, 存在某个 N 使得

[∏1≤k≤sA5 2-生成]⟺[1≤s≤n].

先证明 N 存在. 根据鸽笼原理, 对任意 s≥|A5|2, 某两个 index 上的两组生成元相同, 例如

⋯i0⋯j0⋯⋯(123)⋯(123)⋯⋯(12345)⋯(12345)⋯

从而生成群中不含形如

⋯i0⋯j0⋯⋯(123)⋯(1)⋯

的元素. 显然

[∏1≤k≤sA5 2-生成]⟸[∏1≤k≤s+1A5 2-生成],

从而这样的 N 存在, 粗略的上界为 |A5|2.

再注意到

⋯i0⋯j0⋯⋯(12)(34)⋯(23)(15)⋯⋯(12345)⋯(12345)⋯

显然 2-生成群

⟨(12)(34)×(12345),(23)(15)×(12345)⟩

的大小为 60 而非 602.

方便起见, 称以上的 (12)(45)(12345) 与 (23)(15)(12345) 不互质. 若 ab 与 cd 互质, 当且仅当

⟨a×c,b×d⟩≃A5×A5.

若 [2][2] 为 A5 的生成元, 则其一定是 (ij)(kl)(jl)(im) 的形式换言之, 下方对换形如'上方两个对换中各取一元生成的对换'+'上方对换中任一元与第五元生成的对换'. 从而任意两个形如 [2][2] 的生成元不互质.

若 [2][3] 为 A5 的生成元, 则其一定是 (ij)(kl)(ikm) 的形式. 因为下方 3-轮换一定与上方每个对换以及第五元相交. 从而任意两个形如 [2][3] 的生成元不互质.

若 [3][3] 是 A5 的生成元, 则其一定是 (ijk)(kml) 的形式. 因为用两个 3-轮换覆盖五个元素的方式仅此一类. 从而任意两个形如 [3][3] 的生成元不互质.

若 [5][2] 是 A5 的生成元, 则其形如以下下三种形式之一.

(12345)(12)(34),(12345)(14)(23),(12345)(13)(24).

下表中第四行由'第三行复合第二行'得到

123(12345)(12345)(12345)(12)(34)(14)(23)(13)(24)[3][2][5]

从而三类形如 [5][2] 的生成元两两互质.

若 [5][3] 是 A5 的生成元, 则其形如以下下两种形式之一.

(12345)(123),(12345)(124),(12345)(132),(12345)(142).

下表中第四行由'第三行复合第二行'得到, 第五行由'第三行复合第四行'得到

1234(12345)(12345)(12345)(12345)(123)(124)(132)(142)[5][5][3][2][3][2][5][5]

仿照开头处理 ∏1≤k≤4A5 的方式, 这四类形如 [5][3] 的生成元两两互质.

若 [5][5] 是 A5 的生成元, 则其形如以下下三种形式之一.

(12345)(13245),(12345)(12534),(12345)(15423),(12345)(14352).

下表中第四行由'第二行复合第三行'得到, 第五行由'第二行复合第四行'得到

1234(12345)(12345)(12345)(12345)(13245)(12534)(15423)(14352)[2][5][3][3][5][3][3][2]

仿照开头处理 ∏1≤k≤4A5 的方式, 这四类形如 [5][5] 的生成元两两互质.

综上, [2][2] 共计一类, [2][3] 共计一类, [3][2] 共计一类, [3][3] 共计一类, [5][2] 共计三类, [2][5] 共计三类, [5][3] 共计四类, [3][5] 共计四类, [5][5] 共计四 类. 从而 N=22.
